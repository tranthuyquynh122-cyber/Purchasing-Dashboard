## ðŸ“Œ Background & Overview
This project analyzes purchasing performance data using an interactive Power BI dashboard.  
The goal is to help procurement and operations teams monitor purchasing trends, evaluate vendor performance, and support data-driven purchasing decisions.

**Tools:** Power BI, DAX, Excel / CSV  
**Domain:** Purchasing Analytics / Procurement / Operations  


## ðŸ“‚ Dataset Description & Data Structure
The dataset includes purchasing transactions with details on:
- purchase order dates  
- vendor information  
- product and category data

- total purchase orders

- purchasing trends over time  
- performance by vendor and product  

The data was cleaned and standardized to ensure reliable analysis.
